# CricPulse — HCI-Driven Cricket Platform

> A comprehensive redesign of CricBuzz demonstrating **40+ HCI concepts** from the complete course syllabus.

## 🏏 Live Demo

Open `index.html` in any browser, or deploy to GitHub Pages for public access.

## 📁 Project Structure

```
pro/
├── index.html           → Home / Live Scores Dashboard
├── match-detail.html    → Live Scorecard & Commentary
├── news.html            → News Feed (F-Pattern layout)
├── schedule.html        → Match Schedule (date-grouped)
├── standings.html       → IPL Points Table
├── player.html          → Player Profile (Virat Kohli)
├── search.html          → Search & Filter
├── videos.html          → Video Highlights
├── fantasy.html         → Fantasy Predictor (NEW feature)
├── help.html            → Help & Support + CricBot (NEW)
├── settings.html        → Accessibility & Preferences (NEW)
├── notifications.html   → Notifications Center
├── css/styles.css       → Global Design System
├── js/app.js            → Global JavaScript
└── README.md            → This file
```

## 🎯 HCI Concepts Covered

| Category | Concepts |
|---|---|
| **Nielsen's 10 Heuristics** | H1-H10 all mapped to features |
| **Shneiderman's 8 Rules** | Consistency, Feedback, Closure, Reversal, Shortcuts, etc. |
| **Gestalt Laws** | Proximity, Similarity, Continuity, Closure, Figure-Ground, Common Fate |
| **Classical Laws** | Fitts', Hick's, Tesler's, Miller's, Serial Position, Inverted Pyramid |
| **Design Laws** | Vital Few (80-20), Law of Learning, Mental Models, Closure |
| **Persuasion (Cialdini)** | Scarcity, Social Proof, Authority, Reciprocity, Commitment, Liking |
| **Visual Design** | Balance, Scale, Dominance, Contrast, Rhythm |
| **Color Psychology** | Green=live, Red=wickets, Blue=trust, Orange=highlights, Gold=awards |
| **Shape Psychology** | Rounded cards=friendly, Sharp badges=urgency |
| **Universal Design** | 7 Principles, WCAG/POUR, Accessibility controls |
| **User Support** | FAQ, CricBot chatbot, Tooltips, Onboarding, Contextual help |
| **Navigation Design** | Breadcrumbs, Sticky nav, Back-button support, Wayfinding |
| **Interaction Paradigms** | Direct Manipulation, Form-fill, Command (keyboard), Conversational UI |
| **Web Usability** | F-Pattern, Above-the-fold, Semantic HTML, Responsive design |
| **Asimov's Laws** | Safe data handling, Confirmation before destructive actions |

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `?` | Show shortcuts help |
| `D` | Toggle dark/light mode |
| `/` | Focus search bar |
| `H` | Go to home page |

## 🚀 Deployment (GitHub Pages)

```bash
git init
git add .
git commit -m "CricPulse HCI Project"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/cricpulse.git
git push -u origin main
# Then enable GitHub Pages in repo Settings → Pages → Source: main
```
